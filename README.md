# NBA Hidden Value Analysis

## Project Overview
This project analyzes NBA player statistics to identify potentially underutilized rotation players.

Using Python and statistical analysis, player performance was normalized using per-minute efficiency metrics to allow fair comparison between players with different playing times.

A composite metric called the **Hidden Value Score** was developed to measure overall player efficiency based on scoring, playmaking, and rebounding impact.

## Key Question
Which NBA players generate strong statistical production despite receiving limited playing time?

## Methodology
- Data cleaning and preprocessing in Python
- Creation of per-minute performance metrics
- Development of a composite Hidden Value Score
- Visualization of efficiency vs playing time using scatter plots

## Key Insight
Several rotation players averaging **10–20 minutes per game** demonstrate efficiency levels comparable to players receiving significantly more playing time, suggesting potential hidden value within team rotations.
